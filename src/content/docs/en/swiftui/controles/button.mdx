## Common Modifiers

| Modifier | Description |
|---|---|
| `.buttonStyle(.bordered)` | Bordered style |
| `.tint(.blue)` | Accent color |
| `.disabled(true)` | Disable button |
| `.controlSize(.large)` | Control size |
| `.buttonBorderShape(.capsule)` | Capsule shape |

:::tip
Use `role: .destructive` for actions that delete data. SwiftUI will automatically display the button in red.
:::

## Full Example

<Tabs syncKey="lang">
  <TabItem label="Swift">
    ```swift
    struct CounterView: View {
        @State private var count = 0

        var body: some View {
            VStack(spacing: 20) {
                Text("Count: \(count)")
                    .font(.largeTitle)

                HStack(spacing: 16) {
                    Button("Subtract") { count -= 1 }
                        .buttonStyle(.bordered)
                        .tint(.red)

                    Button("Add") { count += 1 }
                        .buttonStyle(.borderedProminent)
                        .tint(.green)
                }

                Button("Reset", role: .destructive) { count = 0 }
                    .disabled(count == 0)
            }
        }
    }
    ```
  </TabItem>
  <TabItem label="React (Next.js)">
    ```tsx
    "use client";
    import { useState } from "react";

    export default function Counter() {
      const [count, setCount] = useState(0);

      return (
        <div className="flex flex-col items-center gap-5">
          <p className="text-4xl font-bold">Count: {count}</p>

          <div className="flex gap-4">
            <button
              onClick={() => setCount(count - 1)}
              className="rounded-lg border border-red-500 px-4 py-2 text-red-500 hover:bg-red-50"
            >
              Subtract
            </button>
            <button
              onClick={() => setCount(count + 1)}
              className="rounded-lg bg-green-500 px-4 py-2 text-white hover:bg-green-600"
            >
              Add
            </button>
          </div>

          <button
            onClick={() => setCount(0)}
            disabled={count === 0}
            className="text-red-500 hover:underline disabled:text-gray-400 disabled:no-underline"
          >
            Reset
          </button>
        </div>
      );
    }
    ```
  </TabItem>
</Tabs>
