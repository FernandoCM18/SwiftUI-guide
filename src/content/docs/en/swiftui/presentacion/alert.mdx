The `.alert` modifier displays an alert dialog over the current content, useful for confirming important actions.

## Preview

![Alert in SwiftUI](../../../../../assets/swiftui/alert.svg)

## Basic Usage

```swift
struct AlertExample: View {
    @State private var showAlert = false

    var body: some View {
        Button("Delete") { showAlert = true }
        .alert("Delete item?", isPresented: $showAlert) {
            Button("Cancel", role: .cancel) { }
            Button("Delete", role: .destructive) {
                // Perform deletion
            }
        } message: {
            Text("This action cannot be undone.")
        }
    }
}
```

## Alert with TextField

```swift
@State private var name = ""

.alert("New name", isPresented: $show) {
    TextField("Name", text: $name)
    Button("Cancel", role: .cancel) { }
    Button("Save") { }
}
```

> [Try in Swift Playground →](https://swiftfiddle.com/)

:::caution
Alerts interrupt the user's flow. Only use them for important or irreversible actions. For informational messages, consider banners or toasts.
:::

## Full Example

```swift
struct TaskListView: View {
    @State private var tasks = ["Buy groceries", "Study Swift", "Exercise"]
    @State private var showDelete = false
    @State private var deleteIndex: IndexSet?

    var body: some View {
        NavigationStack {
            List {
                ForEach(tasks, id: \.self) { Text($0) }
                .onDelete { indices in
                    deleteIndex = indices
                    showDelete = true
                }
            }
            .navigationTitle("Tasks")
            .alert("Delete task?", isPresented: $showDelete) {
                Button("Cancel", role: .cancel) { deleteIndex = nil }
                Button("Delete", role: .destructive) {
                    if let i = deleteIndex { tasks.remove(atOffsets: i) }
                }
            } message: {
                Text("You won't be able to recover this task.")
            }
        }
    }
}
```
