## Resizing and Scaling

<Tabs syncKey="lang">
  <TabItem label="Swift">
    ```swift
    VStack {
        // Fit: maintains aspect ratio inside frame
        Image(systemName: "photo")
            .resizable()
            .aspectRatio(contentMode: .fit)
            .frame(width: 150, height: 150)

        // Fill: fills the frame, may clip
        Image(systemName: "photo")
            .resizable()
            .aspectRatio(contentMode: .fill)
            .frame(width: 150, height: 100)
            .clipped()
    }
    ```
  </TabItem>
  <TabItem label="React">
    ```tsx
    import Image from "next/image";

    export default function ResizingScaling() {
      return (
        <div className="flex flex-col items-center gap-5">
          {/* Fit: image fits inside the container */}
          <div className="relative size-[150px] bg-gray-200">
            <Image src="/photo.jpg" alt="Fit" fill className="object-contain" />
          </div>
          {/* Fill: image fills the container, may clip */}
          <div className="relative h-[100px] w-[150px] overflow-hidden bg-gray-200">
            <Image src="/photo.jpg" alt="Fill" fill className="object-cover" />
          </div>
        </div>
      );
    }
    ```
  </TabItem>
</Tabs>

## Common Modifiers

| Modifier | Description |
|---|---|
| `.resizable()` | Makes the image resizable |
| `.aspectRatio(contentMode:)` | .fit or .fill |
| `.frame(width:height:)` | Set dimensions |
| `.clipShape(Circle())` | Clip to shape |
| `.overlay()` | Add overlay |
| `.shadow(radius:)` | Add shadow |

:::tip
Always call `.resizable()` before `.frame()`. Without it, the image renders at its native size.
:::

## Full Example

<Tabs syncKey="lang">
  <TabItem label="Swift">
    ```swift
    struct AvatarView: View {
        var body: some View {
            VStack {
                Image(systemName: "person.circle.fill")
                    .resizable()
                    .aspectRatio(contentMode: .fit)
                    .frame(width: 120, height: 120)
                    .foregroundStyle(.blue)
                    .shadow(radius: 5)

                Text("John Doe")
                    .font(.title2)
                    .bold()
            }
        }
    }
    ```
  </TabItem>
  <TabItem label="React">
    ```tsx

      return (
        <div className="flex flex-col items-center">
          <UserCircle className="size-[120px] text-blue-500 drop-shadow-md" />
          <p className="text-2xl font-bold">John Doe</p>
        </div>
      );
    }
    ```
