## Text Modifiers

<Tabs syncKey="lang">
  <TabItem label="Swift">
    ```swift
    VStack(alignment: .leading, spacing: 12) {
        Text("Large Title")
            .font(.largeTitle)

        Text("Bold and Blue")
            .bold()
            .foregroundStyle(.blue)

        Text("Italic Caption")
            .font(.caption)
            .italic()

        Text("Custom Font")
            .font(.system(size: 24, weight: .heavy, design: .rounded))
    }
    ```
  </TabItem>
  <TabItem label="React (Next.js)">
    ```tsx
    export default function TextModifiers() {
      return (
        <div className="flex flex-col items-start gap-3">
          <p className="text-4xl font-bold">Large Title</p>
          <p className="font-bold text-blue-500">Bold and Blue</p>
          <p className="text-xs italic">Italic Caption</p>
          <p
            className="text-2xl font-extrabold"
            style={{ fontFamily: "ui-rounded, system-ui" }}
          >
            Custom Font
          </p>
        </div>
      );
    }
    ```
  </TabItem>
</Tabs>

## Multiline Text

<Tabs syncKey="lang">
  <TabItem label="Swift">
    ```swift
    Text("This is a long paragraph that will wrap across multiple lines automatically when it exceeds the available width.")
        .multilineTextAlignment(.center)
        .lineLimit(3)
        .lineSpacing(4)
    ```
  </TabItem>
  <TabItem label="React (Next.js)">

          automatically when it exceeds the available width.
        </p>
      );

## Common Modifiers

| Modifier | Description |
|---|---|
| `.font(.title)` | Set font style |
| `.bold()` | Bold text |
| `.italic()` | Italic text |
| `.foregroundStyle(.red)` | Text color |
| `.multilineTextAlignment(.center)` | Multi-line alignment |
| `.lineLimit(2)` | Maximum number of lines |
| `.strikethrough()` | Strikethrough |
| `.underline()` | Underline |

:::tip
Combine multiple `Text` views with `+` to create attributed strings: `Text("Hello ").bold() + Text("World").foregroundStyle(.blue)`
:::

## Full Example

<Tabs syncKey="lang">
  <TabItem label="Swift">
    ```swift
    struct WelcomeView: View {
        let username = "Developer"

        var body: some View {
            VStack(spacing: 16) {
                Text("Welcome!")
                    .font(.largeTitle)
                    .bold()

                Text("Hello, \(username)")
                    .font(.title2)
                    .foregroundStyle(.secondary)

                Text("Start building amazing apps with SwiftUI. This framework makes it easy to create beautiful interfaces.")
                    .font(.body)
                    .multilineTextAlignment(.center)
                    .foregroundStyle(.secondary)
                    .padding()
            }
            .padding()
        }
    }
    ```
  </TabItem>
  <TabItem label="React (Next.js)">
    ```tsx
    export default function WelcomeView() {
      const username = "Developer";

      return (
        <div className="flex flex-col items-center gap-4 p-4">
          <h1 className="text-4xl font-bold">Welcome!</h1>
          <p className="text-2xl text-gray-500">Hello, {username}</p>
          <p className="text-center text-gray-500">
            Start building amazing apps with SwiftUI. This framework makes it
            easy to create beautiful interfaces.
          </p>
        </div>
      );
    }
    ```
  </TabItem>
</Tabs>
