**SwiftUI** is Apple's modern declarative framework for building user interfaces across all Apple platforms. Instead of step-by-step instructions, you describe what the UI should look like and SwiftUI handles the rest.

## Basic Structure

```swift
import SwiftUI

@main
struct MyApp: App {
    var body: some Scene {
        WindowGroup {
            ContentView()
        }
    }
}

struct ContentView: View {
    var body: some View {
        Text("Hello, World!")
    }
}
```

> [Try in Swift Playground →](https://swiftfiddle.com/)

## Key Concepts

- **Declarative syntax**: Describe what you want, not how to build it
- **State-driven**: The UI automatically updates when state changes
- **Composable**: Build complex UIs by combining small, reusable views
- **Cross-platform**: Same code works on iOS, macOS, watchOS, and tvOS
