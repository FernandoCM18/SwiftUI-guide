**Combine** is Apple's framework for reactive programming. It lets you process values over time declaratively, handling asynchronous events and data streams.

## What is Reactive Programming?

Instead of actively pulling data, you **subscribe** to a stream and receive data when it's available (push-based).

```swift
import Combine

let publisher = [1, 2, 3, 4, 5].publisher

publisher
    .filter { $0 % 2 == 0 }
    .sink { value in
        print("Received: \(value)")
    }
// Prints: Received: 2, Received: 4
```

> [Try in Swift Playground →](https://swiftfiddle.com/)

## Core Concepts

1. **Publishers**: Emit values over time
2. **Subscribers**: Receive and process those values
3. **Operators**: Transform values between publisher and subscriber
