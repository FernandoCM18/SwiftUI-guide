A **Subscriber** receives values emitted by a Publisher. Combine provides two built-in subscribers: `sink` and `assign`.

## sink

Receives values and completion with closures:

```swift
import Combine

let numbers = [1, 2, 3, 4, 5].publisher

let cancellable = numbers.sink { completion in
    print("Done: \(completion)")
} receiveValue: { value in
    print("Value: \(value)")
}
```

> [Try in Swift Playground →](https://swiftfiddle.com/)

## assign

Assigns values directly to an object property:

```swift
class MyModel: ObservableObject {
    @Published var text = ""
}

let model = MyModel()
let cancellable = ["Hello", "World"].publisher
    .assign(to: \.text, on: model)
```

> [Try in Swift Playground →](https://swiftfiddle.com/)

### assign(to:) with @Published

Optimized version that avoids retain cycles:

```swift
class SearchViewModel: ObservableObject {
    @Published var query = ""
    @Published var results: [String] = []

    init() {
        $query
            .debounce(for: .milliseconds(300), scheduler: RunLoop.main)
            .map { ["Result for: \($0)"] }
            .assign(to: &$results)
    }
}
```

> [Try in Swift Playground →](https://swiftfiddle.com/)

## AnyCancellable

Subscriptions return an `AnyCancellable`. When deallocated, the subscription is automatically cancelled:

```swift
class MyViewModel: ObservableObject {
    private var cancellables = Set<AnyCancellable>()

    func loadData() {
        URLSession.shared.dataTaskPublisher(for: url)
            .map(\.data)
            .decode(type: [Item].self, decoder: JSONDecoder())
            .receive(on: DispatchQueue.main)
            .sink(
                receiveCompletion: { _ in },
                receiveValue: { [weak self] in self?.items = $0 }
            )
            .store(in: &cancellables)
    }
}
```

> [Try in Swift Playground →](https://swiftfiddle.com/)

:::caution
Always store subscriptions in a property or `Set<AnyCancellable>`. If the `AnyCancellable` is released, the subscription is cancelled immediately.
:::

## Summary

| Subscriber | Use | Return |
|---|---|---|
| `sink` | Process values with closures | `AnyCancellable` |
| `assign(to:on:)` | Assign to KVO property | `AnyCancellable` |
| `assign(to:)` | Assign to @Published | Void (self-managed) |
