## ¿Por qué esta guía?

- **Todo en español**: Explicaciones claras y comentarios de código en español.
- **Ejemplos prácticos**: Cada concepto incluye código que puedes probar de inmediato.
- **Comparaciones React / Next.js**: Si vienes de React + TypeScript, cada ejemplo muestra el equivalente en Next.js + Tailwind para que la transición sea más fácil.
- **Progresión natural**: Desde los conceptos básicos hasta patrones avanzados.
- **Visuales incluidos**: Diagramas SVG para los componentes de SwiftUI.
