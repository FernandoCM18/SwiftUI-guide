title: Subscribers

Un **Subscriber** recibe valores emitidos por un Publisher. Combine ofrece dos suscriptores integrados principales: `sink` y `assign`.

## sink

Recibe valores y la finalización con closures:

```swift
import Combine

let numeros = [1, 2, 3, 4, 5].publisher

let cancelable = numeros.sink { completion in
    print("Finalizado: \(completion)")
} receiveValue: { valor in
    print("Valor: \(valor)")
}
```

> [Probar en Swift Playground →](https://swiftfiddle.com/)

### sink solo con valores (Never error)

Cuando el publisher no puede fallar (`Never`), puedes usar la versión simplificada:

```swift
let cancelable = Just("Hola")
    .sink { valor in
        print(valor) // "Hola"
    }
```

> [Probar en Swift Playground →](https://swiftfiddle.com/)

## assign

Asigna valores directamente a una propiedad de un objeto:

```swift
class MiModelo: ObservableObject {
    @Published var texto = ""
}

let modelo = MiModelo()

let cancelable = ["Hola", "Mundo"].publisher
    .assign(to: \.texto, on: modelo)

print(modelo.texto) // "Mundo"
```

> [Probar en Swift Playground →](https://swiftfiddle.com/)

### assign(to:) con @Published

Versión optimizada que evita ciclos de retención:

```swift
class BuscadorViewModel: ObservableObject {
    @Published var busqueda = ""
    @Published var resultados: [String] = []

    init() {
        $busqueda
            .debounce(for: .milliseconds(300), scheduler: RunLoop.main)
            .map { texto in
                // Simular búsqueda
                ["Resultado para: \(texto)"]
            }
            .assign(to: &$resultados)
    }
}
```

## AnyCancellable

Las suscripciones devuelven un `AnyCancellable`. Cuando se libera de memoria, la suscripción se cancela automáticamente:

```swift
class MiViewModel: ObservableObject {
    private var cancellables = Set<AnyCancellable>()

    func cargarDatos() {
        URLSession.shared.dataTaskPublisher(for: url)
            .map(\.data)
            .decode(type: [Elemento].self, decoder: JSONDecoder())
            .receive(on: DispatchQueue.main)
            .sink { completion in
                if case .failure(let error) = completion {
                    print("Error: \(error)")
                }
            } receiveValue: { [weak self] elementos in
                self?.elementos = elementos
            }
            .store(in: &cancellables) // Almacenar suscripción
    }
}
```

> [Probar en Swift Playground →](https://swiftfiddle.com/)

## Cancelar suscripciones

```swift
let cancelable = timer.sink { valor in
    print(valor)
}

// Cancelar manualmente
cancelable.cancel()
```

> [Probar en Swift Playground →](https://swiftfiddle.com/)

:::caution
Siempre almacena las suscripciones en una propiedad o en un `Set<AnyCancellable>`. Si el `AnyCancellable` se libera, la suscripción se cancela inmediatamente.
:::

## Resumen

| Subscriber | Uso | Retorno |
|---|---|---|
| `sink` | Procesar valores con closures | `AnyCancellable` |
| `assign(to:on:)` | Asignar a propiedad KVO | `AnyCancellable` |
| `assign(to:)` | Asignar a @Published | Void (autogestión) |

:::tip
Usa `store(in: &cancellables)` para almacenar múltiples suscripciones de forma limpia en un `Set<AnyCancellable>`.
:::
