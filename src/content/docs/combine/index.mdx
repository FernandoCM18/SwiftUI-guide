**Combine** es el framework de Apple para programación reactiva. Permite procesar valores a lo largo del tiempo de forma declarativa, manejando eventos asíncronos y flujos de datos.

## ¿Qué es la programación reactiva?

La programación reactiva se basa en flujos de datos y la propagación de cambios. En lugar de pedir datos activamente (pull), te **suscribes** a un flujo y recibes datos cuando están disponibles (push).

```swift
import Combine

// Un publisher que emite valores
let publisher = [1, 2, 3, 4, 5].publisher

// Un subscriber que recibe valores
publisher
    .filter { $0 % 2 == 0 }
    .sink { valor in
        print("Recibido: \(valor)")
    }
// Imprime: Recibido: 2, Recibido: 4
```

> [Probar en Swift Playground →](https://swiftfiddle.com/)

## Conceptos fundamentales

Combine se basa en tres conceptos principales:

1. **Publishers**: Emiten valores a lo largo del tiempo
2. **Subscribers**: Reciben y procesan esos valores
3. **Operators**: Transforman los valores entre publisher y subscriber

```swift
// Publisher → Operator → Subscriber
URLSession.shared.dataTaskPublisher(for: url)  // Publisher
    .map(\.data)                                // Operator
    .decode(type: Respuesta.self, decoder: JSONDecoder()) // Operator
    .sink(                                      // Subscriber
        receiveCompletion: { _ in },
        receiveValue: { respuesta in
            print(respuesta)
        }
    )
```

> [Probar en Swift Playground →](https://swiftfiddle.com/)

## ¿Por qué usar Combine?

- **Código declarativo**: Describe qué hacer, no cómo hacerlo
- **Composición**: Encadena operaciones de forma clara
- **Integración nativa**: Funciona con SwiftUI, URLSession, NotificationCenter
- **Manejo de errores**: Control de errores integrado en el flujo
